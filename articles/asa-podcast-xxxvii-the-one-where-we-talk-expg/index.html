ASA Podcast: XXXVII The One Where We Talk ExpG

Written by crowh

Don’t get too excited about the title. We really don’t go into this nearly as much as we probably should have. We draw a lot of our thoughts from the 11tegen11 article that sets the table nicely for the discussion, and we just kind of build around that. The audio for the first portion is a little maddening. We had a few problems, but everything was cleared up after about the first 20 minutes, so don’t be deterred from finishing this one. There are some good points made by Drew and me, if I may say so, which make it well worth a listen.
